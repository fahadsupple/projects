---
name: Melbourne modifier exception rule
description: When Melbourne-modified keyword has no volume, the unmodified term can be used as a secondary if other factors justify it
type: feedback
---

If a keyword has no Melbourne-specific search volume but the national/unmodified version has volume, the unmodified term CAN be used as a secondary keyword when:
- The service is genuinely offered by the client
- The CPC is high (indicates commercial value)
- There are no better Melbourne-modified alternatives available
- The term is NOT a subset of the primary keyword on the same page

**Why:** Melbourne modifier is preferred for relevance and local intent, but forcing it when there's no volume leaves the secondary slot empty or weak. A nationally-searched term still captures local searchers via Google's location detection.

**How to apply:** Only for secondary keywords, not primary. Always try Melbourne-modified version first. Use the unmodified version as a fallback when: Melbourne version = 0/dash, national version has meaningful volume, and the keyword is clearly relevant to the client's service.

**Entity building:** Unmodified secondary keywords also help build topical entity signals on the page — even low-volume national terms reinforce what the page is about to Google's entity graph. This is a valid reason to include them beyond just search volume.
