3. H2 Audit — All 47 Pages

Screaming Frog picks up the first H2 on each page. Every page on this site returns "Send us a message" as H2-1 — this is the contact form widget heading appearing first in the HTML before content H2s. See observation below.

Site-Wide H2 Observation — Flag for Web PM Screaming Frog shows "Send us a message" as the first H2 on all 47 pages. This means the contact form widget (with its own heading) is rendered in the HTML before the content subheadings. Ideally, content H2 headings should appear before the form widget in the page structure, so they register as the primary H2s. This does not affect rankings significantly, but it is worth tidying for page structure and crawlability. Web PM to review with the developer.
