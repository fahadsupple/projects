2. Issues Found

All 47 pages checked. Issues only — all other pages clean. All 34 approved URLs confirmed present on staging.

Page Element Issue Fix
/suspended-ceilings-richmond/ H2 H2 reads "Richmond Homes" on a commercial page Remove "Homes" — change to "Richmond" only
All 47 pages H2 structure Screaming Frog shows "Send us a message" as the first H2 on every page. This means the contact form widget heading is appearing earlier in the HTML source than the actual content subheadings. Google reads the page top-to-bottom in the HTML — so it sees "Send us a message" as the primary H2, not the content heading (e.g. "Why Choose Danton Developments"). The content H2s exist, but the form widget is coded higher up in the HTML than the page body. Developer to check the page template and move the contact form widget below the main content section in the HTML, so content H2s register first. Alternatively, change the form heading from an H2 to a <p> or <div> tag — it doesn't need to be a heading at all.
/suspended-ceilings-grovedale/ Meta description Says "Transform your dated Grovedale home" — this is a commercial page, not residential Replace "home" with "workspace". E.g. "Transform your Grovedale workspace with modern suspended ceilings…"
