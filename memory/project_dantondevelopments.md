3. "office partitions melbourne" → Dedicated inner pages with keyword in URL win.
4. "partition walls melbourne" → Overlapping SERP with "office partitions" — coexist on same page.
5. "glass office partitions" / "glass partition walls" → Separate glass-specific pages rank.
6. "plasterboard partitions" / "metal stud partitions" → Material/method terms, similar SERP. One page.
7. "ceiling repair melbourne" / "ceiling replacement melbourne" → Share same SERP. One combined page.
