# Keyword Strategy — Core Rules from Practice

## 2 keywords per page is not compulsory — pages can have 1, 2, 3, or more
A page can target 1 keyword only, or 3-4 keywords, as long as they all make sense on that page and each additional keyword would bring real business if ranked.
**Why:** The "2 per page" structure is a planning convention, not a hard rule. Forcing 2 keywords on every page can result in weak secondaries that add no value. Conversely, leaving clean volume on the table because a page "already has 2" wastes opportunity.
**How to apply:** At plan finalisation: (a) if no clean secondary exists, run with primary only rather than forcing a bad secondary; (b) if a 3rd or 4th keyword is closely related, clean SERP, and would generate business — add it to the page. Do not add keywords just to fill slots.

## Third and fourth keywords can be added to existing pages
When reviewing a keyword plan, check whether existing pages could absorb a 3rd or 4th keyword — particularly lounge subcategory pages where many product-type variants exist.
**Why:** A page already optimised for a category (e.g. recliner chairs) naturally has topical authority for closely related terms. Adding a 3rd keyword costs nothing and can capture additional traffic if the SERP is clean and the keyword is genuinely different.
**How to apply:** After building the primary/secondary mapping, review each page for related terms with volume that weren't included. Check SERP and subset rules. If both pass, add as 3rd keyword. Do not exceed 4 keywords per page — topical signal gets diluted beyond that.

## Volume ceiling per keyword (non-homepage pages)
Keep all keywords at ≤ 1,000 monthly searches per keyword, except the homepage where higher-volume entity-building keywords are acceptable.
**Why:** New/weak domains (DR < 20) cannot realistically rank for high-volume terms in a 3-month window. Lower volume = lower competition = faster rankings = faster client results.
**How to apply:** At keyword selection stage, filter out any keyword > 1,000/mo for all pages except homepage. Flag as long-term play if over threshold.

## SERP alignment is mandatory for every keyword pair
Before finalising any primary + secondary pair on a page, check the actual Google SERP for BOTH keywords separately. Only map two keywords to the same page if they return the same (or near-identical) competitive set and page types.
**Why:** Misaligned SERPs split the page's topical signal. A page targeting "dining chairs" and "dining tables" will win neither — they are separate product intents with different competitors.
**How to apply:** Run SERP checks for every pair, not just category-level checks. Record which domains appear in each SERP and compare overlap. If different retailers dominate each SERP, they need separate pages.

## Singular/plural are the same keyword — never pair them
GKP groups singular and plural forms (dining chair = dining chairs). Never use these as primary + secondary on the same page — they contribute zero topical diversity.
**Why:** Same search intent, same SERP, zero differentiation. Wastes the secondary keyword slot.
**How to apply:** Always check if secondary is simply the plural/singular of primary. If yes, replace with a genuinely different keyword.

## Avoid location+country modifier as keyword (e.g. "australia")
Do not use keywords like "dining tables sydney australia" as keyword targets. The "australia" suffix is not a meaningful modifier — it doesn't reflect how searchers actually behave and adds noise to tracking.
**Why:** Searchers in Sydney don't type "sydney australia" — they either use "sydney" or no location at all. Using both creates an unnatural keyword that won't cleanly track.
**How to apply:** Strip "australia" from any Sydney-modified keyword. Use "dining table sydney" not "dining table sydney australia."

## Subset rule — strictly enforced
If keyword A contains all words of keyword B, then B is a subset of A. Never pair them on the same page. Test every pair before finalising.
**Why:** Google reads both keywords on a page and the more specific one dilutes the broader one's signal (and vice versa). A subset pair sends conflicting specificity signals.
**How to apply:** For each pair, ask: does keyword 1 contain all words of keyword 2? And vice versa? If yes to either → reject the pair and find a replacement.

## BDM priority category should dominate the keyword plan
When a BDM brief flags a specific category as client priority (e.g. lounge furniture), allocate the majority of page slots to that category and its subcategories.
**Why:** The client's commercial priority drives the revenue intent. More pages in a category builds topical authority faster, which helps all pages in that category rank sooner.
**How to apply:** Count pages per category after initial plan. If BDM priority is under-represented, consider replacing lower-priority category pages with additional subcategory pages of the priority category.

## Subcategory strategy for ecommerce keyword plans
For ecommerce furniture clients, use subcategory pages (e.g. leather lounges, modular sofas, recliner chairs) rather than broad category pages where possible. Subcategory keywords are less competitive, easier to rank, and more purchase-ready.
**Why:** A searcher for "leather lounges sydney" is closer to purchase than one searching "furniture sydney." Subcategory traffic converts better and is achievable faster on weak domains.
**How to apply:** Map each major category (lounge, bedroom, dining, outdoor) to subcategory pages. Only use broad category keywords on the homepage or main category hub page.

## No "cheap" keywords unless the brand explicitly uses "cheap" positioning
Do not use "cheap [product] [city]" as a keyword target unless the client's website explicitly uses that language and their prices are genuinely cheap.
**Why:** "Cheap" attracts bargain-hunters and undermines brand trust for quality-positioned brands. Triple J's USPs are warranty, quality, and price-beat — not being "cheap."
**How to apply:** At keyword selection, filter out any keyword containing "cheap". Exception: only include if the client's website copy uses the word "cheap" directly and their pricing strategy supports it.

## "Bedroom suite" + city modifier pulls hotel SERPs, not furniture SERPs
Never use "[bedroom suite] [city]" as a furniture keyword. "Bedroom suite sydney" returns Four Seasons, Langham, Park Hyatt — hotel accommodation SERPs entirely. Zero furniture retailers present except as stray results.
**Why:** "Bedroom suite" is ambiguous — Google interprets it as hotel accommodation when combined with a city name. Searchers looking for bedroom furniture sets use different language ("bedroom set", "bedroom furniture", "bedroom package").
**How to apply:** At keyword selection, never pair "bedroom suite" with a city modifier. Also check "bedroom set [city]" SERP before using — may have similar intent contamination.

## "For sale" + location modifier consistently pulls classified/clearance SERPs
Do not use "[product] for sale [city]" as a keyword target. Gumtree, Facebook Marketplace, and discount warehouse results dominate these SERPs.
**Why:** "For sale" signals second-hand or clearance intent. Not aligned with a new ecommerce furniture retailer.
**How to apply:** At keyword selection, exclude any keyword containing "for sale" regardless of volume.

## No "sale" + location modifier — use "for sale" instead
Do not use "[product] sale [city]" as a keyword. Use "[product] for sale [city]" instead.
**Why:** "[product] sale [city]" returns event/discount SERPs (people looking for temporary sales). "[product] for sale [city]" returns commercial availability SERPs — aligned with ecommerce category intent.
**How to apply:** At keyword selection, replace any "[X] sale [location]" with "[X] for sale [location]". Check GKP volume for the "for sale" variant separately.

## Homepage entity-building strategy for competitive terms
When the homepage target keywords are too competitive for a new/weak domain to rank for quickly, place them on the homepage for entity-building purposes only — not as reportable ranking targets.
**Why:** Google still reads the on-page keywords to understand what category and city the business operates in. A furniture retailer that mentions "furniture store sydney" and "furniture showroom sydney" on its homepage gets the entity signal even if it never ranks #1 for those terms. Long-term, as DR grows, the page gains natural authority for those terms.
**How to apply:** For DR < 20 domains with highly competitive homepage keywords (6,000+/mo, national chains dominating SERP): mark them as entity-building only, exclude them from the client's reporting budget, and allocate those keyword slots to a new page where quick rankings are achievable. Make the distinction clear in the HTML deliverable.

## SERP intent contamination — always check what the SERP is actually returning
Before finalising any keyword, check that the SERP returns the right page type (category pages from furniture retailers, not hotels, classifieds, renovation companies, or specialist trades).
**Why:** Many keywords that look correct on paper return completely wrong intent. "Bedroom suite sydney" = hotel accommodation. "Kitchen furniture" = kitchen renovation companies. "Lighting" = electrical specialists. These keywords waste page slots and will never generate furniture sales.
**How to apply:** At keyword selection, always ask: "What type of page does this SERP return?" Furniture retailer category pages = correct. Hotels, Gumtree, renovation companies, trade specialists = wrong intent, reject.

## Specialist SERP vs general furniture SERP — different competitive sets
Some product categories return SERPs dominated by dedicated specialists rather than general furniture retailers. This makes it harder for a general furniture retailer to rank, even without national chains present.
**Why:** "Home office furniture sydney" returns dedicated office furniture specialists (Equip, Elite, Northside). Google has decided these specialists are most relevant for the query. A general furniture retailer with a home office category page is a weaker match for this intent than a store that sells only office furniture.
**How to apply:** At SERP review, check whether the top 5 results are all dedicated specialists in that category (e.g. only office furniture stores, only mattress stores, only kitchen companies). If yes, the keyword is harder for a general retailer even if no national chain is present. Deprioritise in favour of keywords where the SERP is a mix of general furniture retailers.

## Size-specific sofa keywords (3 seater, 2 seater) are dominated by national chains
Keywords like "3 seater sofa sydney", "2 seater sofa sydney" consistently return Freedom, IKEA, Lounge Lovers, King Living. High-volume, high-competition. Not suitable for DR < 20 domains.
**Why:** Size-specific sofa searches have very high volume and intent, which attracts major national brands who invest heavily in these category pages. The SERP is dominated by specialists and national chains at every price point.
**How to apply:** Skip all size-specific sofa keywords (3 seater, 2 seater, 4 seater) for weak domains. Target product-type keywords instead (leather lounges, fabric lounges, chaise lounges) — these are more specific and return local specialist SERPs.

## Bedroom piece keywords at meaningful volume almost always have national chains
Individual bedroom furniture pieces (chest of drawers, bedside tables, tallboys, entertainment units, wardrobes, buffets, storage beds nationally) consistently return Amart, Harvey Norman, Temple & Webster, Freedom, or IKEA at meaningful volume.
**Why:** Bedroom furniture pieces are high-volume, high-margin categories that all major retailers invest in. At any volume worth targeting (50/mo+), the SERP has been claimed by national chains.
**How to apply:** For bedroom categories on weak domains: (a) target broad "bedroom furniture [city]" with a city modifier rather than individual pieces; (b) use very specific product modifiers (storage bed sydney = clean at 40/mo) rather than generic pieces; (c) if individual piece SERPs are all taken by chains, substitute a different product category entirely (e.g. coffee tables).

## QA process — run SERP checks on ALL keyword pairs before finalising any plan
Before any plan is considered final, run a live SERP check on every individual keyword (primary and secondary) in the plan. SERPs change — a keyword that was clean at research time may have a national chain ranking by the time the plan is delivered.
**Why:** During v1.3 QA for triplejfurniture, 4 secondaries that had been previously validated were found to have national chains ranking (IKEA, Amart, Harvey Norman, Freedom) when re-checked. One secondary ("bedroom suite sydney") had completely wrong intent (hotel suites). These would have been delivered to the client without QA.
**How to apply:** After building the keyword mapping, run live search checks on all 20 keywords (or equivalent number). Do not rely on prior research alone. For each keyword, apply the proportion test: are small/independent stores dominant? Replace keywords where chains hold 3+ positions, or where the SERP has no viable independent presence.

## National chain presence — threshold for rejection vs amber
The primary test is not position — it is proportion. 1–2 national chains anywhere in the top 10 is acceptable provided small/independent stores dominate the remaining positions. A SERP where chains hold 3+ spots, or where a single chain holds #1 AND #2, is a rejection signal.

**Why:** A client confirmed during triplejfurniture.com.au validation that 1–2 chains present is fine as long as small stores are winning the majority of the page. The old position-only rule (chains at 1–4 = reject) was too strict — it rejected viable keywords where independents held #1 and #2 and a chain only appeared at #3 or #4.

**How to apply:**
- Count chains vs independents across the full top 10
- If chains ≤ 2 AND independents hold the majority → PASS (or amber if a chain holds #1 or #2)
- If chains ≥ 3 OR a single chain holds both #1 and #2 → REJECT
- Always ask: "Are small stores winning this SERP?" If yes, the client can compete. If no, reject.
- Fantastic Furniture is NOT on the reject list; treat as a beatable mid-tier chain
- Nick Scali and Coco Republic are treat as real chains (reject-eligible) but not as dominant as IKEA/Harvey Norman/Amart

## Keyword + location modifier = same keyword — never pair them on the same page
"[keyword]" and "[keyword] [city]" are the same keyword in different forms. Never map them as primary + secondary on the same page.
**Why:** Google treats "[keyword]" and "[keyword] sydney" as the same topical signal. Pairing them wastes the secondary slot and provides zero topical diversification.
**How to apply:** Before finalising any pair, check: is the secondary simply the primary + a location suffix (or vice versa)? If yes, it is the same keyword — replace the secondary with a genuinely different term.
