⚠ Critical Fixes — Do These First

These 4 issues cause the most immediate damage to traffic and rankings. All are fixable within 1 hour.

  • Critical
    Missing 301 redirects — 1,410+ clicks/90 days going to 404 errors
    The site migrated URLs but never set up redirects. GSC still shows traffic to the old URLs.
    Add these 4 redirects in .htaccess or LiteSpeed rewrite rules:
    /gosford/ → /service-areas/gosford/
    /centrelink-loans/ → /loans/loans-for-people-on-centrelink/
    /tweed-heads/ → /service-areas/tweed-heads/
    /wollongong/ → /service-areas/wollongong/
  • Critical
    HTTP does not redirect to HTTPS
    http://surecash.com.au/ serves the site without redirecting to HTTPS. Creates HTTP/HTTPS duplicate content split and serves traffic insecurely.
    Add to .htaccess: RewriteCond %{HTTPS} off / RewriteRule ^ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
  • Critical
    4 internal/admin pages are publicly indexed
    /leads/, /citations/, /brand-page/, /lm-buy/ are indexable with no noindex tag. These appear in Google's index and dilute crawl budget.
    Fix: Add <meta name="robots" content="noindex, nofollow"> to each page via Yoast SEO page settings, or add to robots.txt.
  • Critical
    WordPress user enumeration via REST API — security risk
    /wp-json/wp/v2/users publicly exposes usernames (keiran_kirannyyy, colourfaster_pietzzz, frogs-n-dogs). These are attack surface for credential brute-force.
    Fix: Add to functions.php or a security plugin: add_filter('rest_endpoints', function($e){ unset($e['/wp/v2/users'], $e['/wp/v2/users/(?P<id>[\d]+)']); return $e; });

Duplicate Content

Virtual Service Area Pages (98%+ duplicate)

16+ service area pages for cities without a physical branch are near-identical copies of each other. Only the city name changes. Examples: Gosford vs Albury = 98.3% similar; Gosford vs Dubbo = 98.6% similar. These pages create a duplicate content cluster that signals low quality to Google.

Recommended action: Noindex pages with no GSC traffic. For retained pages, add unique local content: specific service areas served in that city, local Centrelink office info, relevant local financial context.

Loan Amount Pages (92–96% duplicate)

9 pages covering $500 to $5,000 loan amounts are 92–96% identical. Only the dollar figure changes. These pages are indexed but provide minimal unique value.

Recommended action: Add amount-specific content: SACC vs MACC classification, specific repayment tables for each amount over 13/26/39 weeks, use-case examples per amount tier (e.g., "$500 for emergency car repair").

Phase 4 — Local SEO

Google Business Profile, NAP consistency, LocalBusiness schema, and branch page audit.

Strategic Alert: GBP Performance ≠ Organic SEO Performance

SE Ranking shows that tracked keyword positions are predominantly local_pack type (Google Maps / GBP), not organic web rankings. The business is effectively competing in local search via its GBP listings, while its website's organic authority is underdeveloped. Any change to Google's local pack algorithm could eliminate the majority of current "rankings." Building organic website authority is essential for resilience.

Virtual Service Area Pages — Local Pack Risk

The site has 26+ service area pages for cities where it has no physical branch (Melbourne, Adelaide, Cairns, Geelong, etc.). These pages cannot appear in the local pack (Google Maps) for these cities as the business has no physical presence there. For organic search, these pages are 98%+ duplicate content with each other, providing minimal organic ranking benefit.

Recommendation: Pull GSC data for each virtual service area page. Noindex any with fewer than 20 organic impressions per month. Retain only those with genuine search demand, and invest in differentiating those pages.
