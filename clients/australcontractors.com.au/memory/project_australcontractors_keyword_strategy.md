## Budget
- 8 general keywords + 24 location keywords = 40 location-equivalent units
- 1 general = 2 location keyword units

## Entity Framework

### Homepage `/`
- Primary entity: Brand (Austral Contractors Group)
- Secondary entity: Concrete Cutting (dominant service)
- Location entity: Melbourne, Victoria
- Signal to Google: "This is the Melbourne concrete cutting specialist company"
- Do NOT try to rank for suburb-level or individual service terms here
- Competitors (megasaw, vicsawing, kustomcut) all use their HOMEPAGES to rank for "concrete cutting melbourne"

### Service Pages
- Primary entity: The specific service (e.g., Core Drilling)
- Location entity: Melbourne
- Supporting entities: related techniques, equipment, materials
- Should be at `/our-services/[service]/` (existing pages) or `/[service]-melbourne/` (new)

### Location Pages (suburb-level)
- Primary entity: Service + Suburb (e.g., Concrete Cutting + Dandenong)
- Secondary entity: Brand proximity signals
- Supporting: suburb name, LGA, nearby suburbs
- URL pattern: `/concrete-cutting-[suburb]/` and `/core-drilling-[suburb]/`

## General Keywords → Page Mapping (8 keywords, 4 pages, 2 per page)

### Page 1: Homepage `/`
- `concrete cutting melbourne` (Ahrefs 300 vol) — PRIMARY
- `concrete cutting services` (Ahrefs 450 vol) — SECONDARY
- SERP check: Both show contractor homepages + service pages, both transactional, local pack YES
- Pairing verdict: GOOD — SERPs heavily overlap

### Page 2: Core Drilling Melbourne → existing `/our-services/concrete-core-drilling/`
- `core drilling melbourne` (Ahrefs 150 vol) — PRIMARY
- `concrete coring` (Ahrefs 150 vol) — SECONDARY
- SERP check: #1 has dedicated inner pages dominating (e.g., /core-drilling-melbourne/). #2 has mixed intent (commercial + some informational, eBay for tools appeared)
- Pairing verdict: OK — #2 has some informational layer, include "what is concrete coring" paragraph on page
- NOTE: Competitors use dedicated `/core-drilling-melbourne/` inner pages (not homepages) for this term. Strong signal to create/optimize this specific page.

### Page 3: Concrete Removal → existing `/our-services/demolition-concrete-removal/`
- `concrete removal` (Ahrefs 500 vol) — PRIMARY
- `concrete demolition and removal` (Ahrefs 100 vol) — SECONDARY
- SERP check: Both transactional, local pack YES. IMPORTANT: Competitive set for "concrete removal" includes concreters, demolition companies, excavation firms — NOT just concrete cutting specialists.
- Pairing verdict: GOOD — SERPs overlap
- FLAG: Content must address removal via sawing/robotic demolition (Austral Contractors' method), not traditional excavation. Differentiation opportunity.

### Page 4: Sawing Services → existing `/our-services/concrete-cutting-sawing/` or new `/wall-sawing-melbourne/`
- `wall sawing` (Ahrefs 150 vol) — PRIMARY
- `ring sawing` (Ahrefs 90 vol) — SECONDARY
- SERP check: Both show specialist service pages. No local pack. Very thin competition (few dedicated pages exist = easy ranking).
- Pairing verdict: GOOD — nearly identical SERPs, very similar niche audience

## Location Keywords (24 keywords = 2 roots × 12 locations)

### Root 1: `concrete cutting`
Pages: `/concrete-cutting-[suburb]/` × 12 suburbs

### Root 2: `core drilling`
Pages: `/core-drilling-[suburb]/` × 12 suburbs

### Why these 2 roots
- Strongest location-variant volumes in Ahrefs data: concrete cutting Melbourne=300, core drilling Melbourne=150
- Other service terms have very weak Melbourne variants (wall sawing Melbourne=10, ring sawing Melbourne=20)
- 12 suburbs per root gives good Melbourne metro + Bendigo coverage

### Suburbs to confirm via GKP (candidates)
Melbourne metro: Dandenong, Frankston, Cranbourne, Pakenham, Werribee, Ringwood, Taylors Lakes, Sunbury, Point Cook, Clayton
Regional VIC: Bendigo, Geelong, Ballarat

## SERP Key Learnings

1. "concrete cutting melbourne" → Homepages dominate (megasaw, kustomcut, vicsawing all homepage) → use homepage
2. "core drilling melbourne" → Dedicated inner pages dominate (coreandsaw.com.au/core-drilling-melbourne/, diacore.com.au/...) → must have dedicated service page
3. "concrete removal" → Competitive set is BROADER than just concrete cutters. Includes concreters, demolition firms, excavation companies.
4. "core drilling" (no location) → Ambiguous SERP — splits between concrete drilling and mining/exploration core drilling. Melbourne modifier is essential.
5. "concrete cutting" (no location) → Same SERP as "concrete cutting melbourne" for Melbourne IP users. Not a separate target.
6. Wall sawing + ring sawing → Very thin competition, almost no dedicated pages. Easy ranking. Niche B2B audience (tradespeople, not homeowners).
